**MCQCard.tsx:**
- State: `selectedIndex: number | null`, `result: EvaluationResult | null`, `submitted: boolean`
- Get lang from `useLanguage()` hook
- Render `exercise.prompt[lang]` as question heading
- Render `exercise.options.map()` as radio-button-style list items: each is a button with `role="radio"`, `aria-checked`, border highlight on selection
- Submit button: disabled when selectedIndex is null or submitted is true
- On submit: call `evaluateMCQ(exercise, selectedIndex!)`, set result, set submitted=true, call `onComplete({ exerciseId: exercise.id, score: result.score, passed: result.passed })`
- When result exists, render `<FeedbackCard result={result} exercise={exercise} lang={lang} />`
- Styling: options as rounded cards with `border-2`, selected gets `border-indigo-500 bg-indigo-50`, hover gets `border-gray-300`

**PickBetterCard.tsx:**
- State: `selected: 'A' | 'B' | null`, `result: EvaluationResult | null`, `submitted: boolean`
- Render prompt, then two option cards in a flex row (`flex flex-wrap gap-4`, each `flex-1 min-w-[140px]`)
- Each option card shows the option text, has `role="radio"`, `aria-checked`
- Selected card gets `ring-2 ring-indigo-500`
- Submit + evaluate via `evaluatePickBetter(exercise, selected!)`
- Show FeedbackCard, call onComplete

**FreeTextCard.tsx:**
- State: `answer: string`, `result: EvaluationResult | null`, `submitted: boolean`
- Render prompt

- Textarea with `rows={4}`, controlled by answer state, placeholder from `exercise.hint?.[lang]`
- Submit disabled when `answer.trim().length === 0` or submitted
- On submit: `evaluateFreeText(exercise, answer, lang)`
- Show FeedbackCard (will include breakdown since evaluator returns it)
- Call onComplete
