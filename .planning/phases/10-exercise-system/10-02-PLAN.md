- State: `answer: string`, `result: EvaluationResult | null`, `submitted: boolean`

- On submit: `evaluateFillBlank(exercise, answer, lang)`, show FeedbackCard, call onComplete

**SpotProblemCard.tsx:**
- State: `selectedIndices: Set<number>`, `result: EvaluationResult | null`, `submitted: boolean`
- Get lang from `useLanguage()`
- Combine `exercise.issues` and `exercise.distractors` into a single array. Shuffle with a deterministic seed based on `exercise.id` (simple hash: sum of charCodes mod length, use Fisher-Yates with seeded random). Store the mapping from shuffled index to original index.
- Render `exercise.problematicPrompt[lang]` in a styled blockquote (same amber border-s-4 pattern)
- Render the combined list as checkboxes: each is a label with checkbox input + text. Selected items get `bg-indigo-50 border-indigo-500`
- Submit disabled when no selections or submitted
- On submit: convert selectedIndices back to original indices, call `evaluateSpotProblem(exercise, originalIndices, lang)`, show FeedbackCard, call onComplete
- Note: evaluator expects indices where 0..issues.length-1 are correct issues, so must map shuffled selections back

**SimulatedChatCard.tsx:**
- State: `userPrompt: string`, `promptSubmitted: boolean`, `result: EvaluationResult | null`, `submitted: boolean`
- Two-phase UI:
  - Phase 1 (promptSubmitted=false): Show `exercise.systemContext?.[lang]` as context block if present, show `exercise.prompt[lang]` as instruction, textarea for user's prompt, submit button "Send to AI"
  - Phase 2 (promptSubmitted=true, result=null): Show user's prompt in a right-aligned chat bubble (bg-indigo-100), show `exercise.preScriptedResponse[lang]` in a left-aligned chat bubble (bg-gray-100) with a brief typing delay (300ms setTimeout), then auto-evaluate: `evaluateSimulatedChat(exercise, userPrompt, lang)`, set result, call onComplete
  - Phase 3 (result exists): Show the chat bubbles + FeedbackCard below with breakdown
- All chat bubbles use `rounded-2xl` with `ps-4 pe-4 py-3` and max-width 80%
- User bubble: `ms-auto` (pushed to end), AI bubble: `me-auto` (pushed to start)
