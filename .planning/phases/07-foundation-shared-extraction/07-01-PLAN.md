1. Create the web/ project. Use `npm create vite@latest web -- --template react-ts` from the repo root, OR manually create the files (manual is preferred for full control). The key files:

2. `web/package.json`:
   - `"name": "@promptplay/web"`
   - `"private": true`
   - `"type": "module"`
   - Dependencies: `react` ^19, `react-dom` ^19
   - DevDependencies: `@vitejs/plugin-react`, `vite` ^6, `typescript` ~5.7, `@types/react` ^19, `@types/react-dom` ^19, `tailwindcss` ^4, `@tailwindcss/vite`
   - Scripts: `"dev": "vite"`, `"build": "tsc -b && vite build"`, `"preview": "vite preview"`

3. `web/vite.config.ts`:
   ```typescript
   import { defineConfig } from 'vite'
   import react from '@vitejs/plugin-react'
   import tailwindcss from '@tailwindcss/vite'
   import path from 'path'

   export default defineConfig({
     plugins: [
       react(),
       tailwindcss(),
     ],
     resolve: {
       alias: {
         '@': path.resolve(__dirname, 'src'),
         '@shared': path.resolve(__dirname, '../shared/src'),
       },
     },
   })
   ```

4. `web/tsconfig.json` — project references style:
   ```json
   {
     "files": [],
     "references": [
       { "path": "./tsconfig.app.json" },
       { "path": "./tsconfig.node.json" }
     ]
   }
   ```

       "module": "ESNext",
       "skipLibCheck": true,
       "moduleResolution": "bundler",
       "allowImportingTsExtensions": true,
       "isolatedModules": true,
       "moduleDetection": "force",
       "noEmit": true,
       "jsx": "react-jsx",
