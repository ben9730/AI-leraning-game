       State machine with `type LessonPhase = 'intro' | 'running' | 'complete'`:
       - `phase` (LessonPhase) -- starts at 'intro'

       - `exerciseCompleted` (boolean) -- whether current exercise has called onComplete

       Use a `completionHandledRef = useRef(false)` to prevent double-firing store actions in StrictMode.

       **Lesson loading:**
       ```typescript
       const { id } = useParams<{ id: string }>()
       const navigate = useNavigate()
       const lesson = useMemo(() => {
         try { return loadLesson(id!) }

       }, [id])

       // Redirect on invalid ID

       ```

       **Exercise completion handler (useCallback):**
       ```typescript
       const handleExerciseComplete = useCallback((result: ExerciseResult) => {
         setExerciseCompleted(true)
       }, [])
       ```
       NOTE: Do NOT auto-advance here. The user must click "Continue" to advance after reading feedback.

       **Continue handler:**
       ```typescript
       const handleContinue = useCallback(() => {
         const exercises = lesson.exercises.sort((a, b) => a.order - b.order)
         if (exerciseIndex + 1 < exercises.length) {
           setExerciseIndex(i => i + 1)
           setExerciseCompleted(false)
         } else {
           // All exercises done -- fire store actions ONCE
           if (!completionHandledRef.current) {
             completionHandledRef.current = true
             completeLesson(lesson.id)
             // Unlock next lesson
             const allIds = getAllLessonIds()
             const idx = allIds.indexOf(lesson.id)
             if (idx !== -1 && idx + 1 < allIds.length) {
               unlockLesson(allIds[idx + 1])
             }
             addXP(lesson.xpReward, 'lesson_complete')
           }
           setPhase('complete')
         }
       }, [lesson, exerciseIndex, completeLesson, unlockLesson, addXP])
       ```

       import { chapters } from '@/content'

       const nextLessonId = useMemo(() => {
