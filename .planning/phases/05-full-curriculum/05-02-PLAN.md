    - Exercises: (1) fill-blank completing "Write a summary of ___ as a ___" (acceptable: bullet points, numbered list, table), (2) mcq about when to specify format, (3) free-text writing a prompt that specifies output format

    - Exercises: (1) spot-problem identifying what's missing from a weak prompt, (2) simulated-chat where user writes a full prompt combining all 3 elements (rubric checks clarity + context + format keywords)
