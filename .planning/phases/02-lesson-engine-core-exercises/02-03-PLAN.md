    **Part A: FeedbackCard component**

    Create `PromptPlay/src/features/exercise/components/FeedbackCard.tsx`:
    - Props: `{ result: EvaluationResult; rubric?: PromptRubric; modelAnswer?: LocalizedString; lang: 'en' | 'he' }`
    - Renders a card showing:
      1. Pass/fail indicator — green checkmark text for pass, red X text for fail, with the overall score percentage
      2. `result.feedback[lang]` as the main feedback message
      3. If `result.breakdown` exists AND `rubric` is provided: render per-criterion breakdown:
         - For each criterion in `rubric.criteria`, show:
           - `criterion.label[lang]` as the criterion name
           - A colored bar (View with percentage width) — green if score > 0, gray if 0
           - The score value as text (e.g. "50%")
         - Use `paddingStart/End` for all layout — RTL safe
      4. If `modelAnswer` is provided: show it in a styled box labeled `t('exercise.model_answer')` ("One strong version:" / "גרסה חזקה אחת:") — always shown regardless of pass/fail
    - Style: rounded card with subtle border, padding 16, marginTop 12

    **Wire FeedbackCard into FreeTextCard and SpotProblemCard:**
    - In `FreeTextCard.tsx`: after submission, when `result` is set, render `<FeedbackCard result={result} rubric={exercise.rubric} modelAnswer={exercise.modelAnswer} lang={lang} />` between the score display and the Continue button
    - In `SpotProblemCard.tsx`: after submission, render `<FeedbackCard result={result} lang={lang} />` (no rubric/modelAnswer for spot-problem — just feedback text and score)

    **Lesson 02: "Be Clear" (`lesson-02-clarity.json`)**
    - `id: "lesson-02-clarity"`, `order: 2`, `chapter: 1`, `prerequisites: ["lesson-01-what-is-prompting"]`, `xpReward: 10`
    - Content: Teach that clarity means the AI knows exactly what you want. A clear prompt has a specific action verb and a defined output format.
    - Exercise 1 (MCQ): "Which of these prompts is the clearest?" — 3 options ranging from vague ("Help me") to clear ("Write a 3-paragraph blog post about remote work benefits")
    - Exercise 2 (free-text): "Rewrite this vague prompt to be clearer: 'Help me with my email.'" — rubric checks for clarity (keywords: "write", "draft", "create", "compose") and specificity (keywords: "email", "message", "paragraph", "short", "long", "words")
    - Exercise 3 (pick-better): Compare a vague vs clear prompt about getting recipe help

    **Lesson 03: "Be Specific" (`lesson-03-specificity.json`)**
    - `id: "lesson-03-specificity"`, `order: 3`, `chapter: 1`, `prerequisites: ["lesson-02-clarity"]`, `xpReward: 10`
    - Content: Teach that specificity means including details — who, what, how many, what format, what tone.
    - Exercise 1 (MCQ): "What makes a prompt specific?" — test understanding of specificity concept
    - Exercise 2 (fill-blank): Template: "Write a ___ for my ___ about ___" — acceptableAnswers for first blank could be "paragraph", "summary", "list", "email"
    - Exercise 3 (free-text): "Make this prompt more specific: 'Tell me about dogs.'" — rubric checks specificity (keywords: "breed", "type", "specific", "number", "list") and context (keywords: "for", "because", "about", "topic")

    **Lesson 04: "Give Context" (`lesson-04-context.json`)**
    - `id: "lesson-04-context"`, `order: 4`, `chapter: 1`, `prerequisites: ["lesson-03-specificity"]`, `xpReward: 15`
    - Content: Teach that context helps the AI understand your situation — who you are, who the audience is, what you've tried.
    - Exercise 1 (pick-better): Compare a prompt without context vs one with context about writing a cover letter
    - Exercise 2 (free-text): "Add context to this prompt: 'Help me prepare for a meeting.'" — rubric checks context (keywords: "meeting", "team", "manager", "client", "project", "about") and intent (keywords: "prepare", "talking points", "agenda", "notes", "summary")
    - Exercise 3 (MCQ): "Which piece of context would help the AI most?" — test ability to identify useful context

    **Lesson 05: "State Your Intent" (`lesson-05-intent.json`)**
    - `id: "lesson-05-intent"`, `order: 5`, `chapter: 1`, `prerequisites: ["lesson-04-context"]`, `xpReward: 15`
    - Content: Teach that intent tells the AI what you plan to DO with the output — this shapes the format, length, and tone of the response.
    - Exercise 1 (spot-problem): Show a prompt like "Tell me about climate change" — issues: "No specified intent/purpose", "No output format", "Too broad" — distractors: "Too short", "Wrong language"
    - Exercise 2 (free-text): "Rewrite this prompt with clear intent: 'Write something about my product.'" — rubric checks intent (keywords: "description", "review", "ad", "post", "marketing", "sell") and clarity (keywords: "product", "feature", "benefit", "audience", "customer")
    - Exercise 3 (MCQ): "What does stating your intent help the AI do?" — test conceptual understanding
