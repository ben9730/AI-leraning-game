# Requirements: PromptPlay v2.1

**Defined:** 2026-03-30
**Core Value:** Make AI skills accessible and fun for anyone — turn "I don't know what to ask AI" into confident, effective AI usage through gamified practice.

## v2.1 Requirements

Requirements for Practical AI & Sandbox milestone. Each maps to roadmap phases.

### Infrastructure

- [ ] **INFRA-01**: CodeBlock component renders code snippets with `dir="ltr"` for RTL safety in Hebrew mode
- [ ] **INFRA-02**: Content schema extended with optional code-related fields for exercises

### Content

- [ ] **CONT-01**: User can complete Lesson 21 — "Intro to AI-Assisted Coding" (what AI can/can't do for code)
- [ ] **CONT-02**: User can complete Lesson 22 — "Writing Your First Script with AI" (prompt-to-script pattern)
- [ ] **CONT-03**: User can complete Lesson 23 — "Debugging with AI" (describe-the-bug, explain-the-error)
- [ ] **CONT-04**: User can complete Lesson 24 — "Automating Repetitive Tasks" (AI for task automation)
- [ ] **CONT-05**: User can complete Lesson 25 — "Putting It Together" (multi-step coding workflow)
- [ ] **CONT-06**: Chapter 5 lessons use code display with syntax highlighting in exercises

### Sandbox

- [ ] **SAND-01**: User can access Practice Sandbox from navigation
- [ ] **SAND-02**: User can select from pre-built coding scenario templates
- [ ] **SAND-03**: User can type a freeform prompt for the selected scenario
- [ ] **SAND-04**: Sandbox displays simulated AI response matched to scenario and prompt patterns

## Future Requirements

## Out of Scope

| Feature | Reason |
|---------|--------|
| Live AI API in sandbox | Zero-cost constraint; all interactions must be simulated |
| Prompt iteration exercise type | Approximated by pick-better; avoid engine risk this milestone |
| Sandbox XP/scoring | Keep sandbox pressure-free; separate from gamification |
| Multi-language code support | Start with JS/Python snippets only; expand later |
| Leaderboards / social features | Not needed to validate code chapter value |
